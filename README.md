# SuperCode Fake Shop Project

## Purpose of the website

- Create a Fake Shop
- Fetch Data from Fakestore API
- Dynamic Routing with React Router DOM

## Built with

- Mobile Design
- Grid and Flexbox Layout
- React
- React Router DOM
- Fetch-API
